To make this simple concept app work you need:

1) VLC media player correctly installed in your machine.
2) VLC set as the default program to start .mp4 files.
3) VLC web services running. (with the default user, default port, etc)

Steps to achieve number 3.
a) Open VLC -> Tools -> Preferences
b) Show all settings (on the bottom left)
c) Go to Interface -> Main Interfaces tab and check "web".
d) Go to Interface -> Main Interfaces -> Lua tab and set you password.
Remember the password, it's gonna be required by the app.
e) check that the firewall is ok with this when saving.
